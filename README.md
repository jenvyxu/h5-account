# React H5在线记账
![](https://img.shields.io/badge/license-MIT-brightgreen)
![](https://img.shields.io/badge/npm-17.0.1-brightgreen)
## 技术栈
- React
- React-redux
- React-router
- Typescript
- ECharts

## 功能
- 首页
  - 下拉展示最近三天记账记录
  - 今天的收入和支出统计
- 记账页
  - 切换收入/支出
  - 选择标签、新增标签
  - 加减运算
  - 提交按钮
- 统计页
  - 七天收支变化折线图
  - 收入和支出统计图
